Finalidad

La finalidad de este sitio web se puede dividir en partes.

Presentación

Hola! este soy yo o al menos una parte de mi y de lo que puedo llegar a hacer. Esta web parte de un estilo de terceros pero he realizado muchos cambios buscando reflejar un poco de mi personalidad y que se adapte a lo que me gustaría mostrar.

Esta Web

La idea de esta web viene de mucho antes del 2014 aun cuando no tenía casi ningún conocimiento claro de este mundo. A lo largo del tiempo el contenido y el lugar donde se aloja esta información ha cambiado mucho así como el enfoque mismo del sitio.

Curiosidad

Simplemente sentía curiosidad de lo que eran las github pages y lo que podían hacer, así que me anime a probar a la par de que aprendía a usar el cliente de git en escritorio.

Escritos

Este es un sitio bueno para subir mis escritos y diversas cosas más. La razón detrás de esto es que es muy difícil encontrar cierta información de algunos temas pues varias palabras o títulos de publicaciones se suelen superponer a los temas más técnicos y dificulta mucho encontrar estas cosas.

Demostración

Quería mostrar un poco de lo que puedo hacer y que más personas lo vieran. Ya que este alojamiento no soporta códigos más avanzados de programación me pareció una muy buena idea para mostrar un nivel inicial y de paso hacerme de un lugar donde poder mostrar algunas cosas más.
